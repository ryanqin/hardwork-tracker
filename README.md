# 苦功夫追踪器

> 苦功夫 = 刻意练习、高质量专注工作时间。每一分钟都算数。

## 功能

- 📝 **记录苦功夫** — 选择类别、输入时长、可加备注
- 📊 **今日概览** — 总时长 + 各类别分布
- 🔥 **连续打卡** — 当前 streak 和历史最高
- 🗓️ **热力图** — 过去30天日历视图

## 类别

💻 编程 · 📚 学习 · 💪 健身 · ✍️ 写作 · ⚡ 其他

## 开发

```bash
npm install
npm run dev
```

浏览器打开 http://localhost:3000

## 技术栈

- Next.js 14 + TypeScript
- Tailwind CSS
- localStorage（无需后端）
